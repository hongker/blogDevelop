## 查看所有命令

## 常用命令
### 查看docker信息
```
docker info
```

### 查看docker的版本
```
docker version
```

### 拉取镜像
```
docker pull hello-world
```

### 运行容器
根据上面下载的镜像跑一个容器
```
docker run hello-world
```
